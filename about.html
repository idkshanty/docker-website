About This Website

This website was created as part of a Docker containerization exercise.

What is Docker?

Docker is a tool that lets you package an application and run it in a container. This means it can run on any machine without needing to install dependencies manually.

How this site works

The HTML and CSS files are copied into a Docker image. Nginx serves the files on port 80 inside the container, which is mapped to port 8080 on the host machine.
